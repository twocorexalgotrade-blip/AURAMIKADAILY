# Swarna Setu Web Application

A comprehensive web version of the Swarna Setu jewelry shopping platform.

## Features

- Firebase OTP Authentication
- Browse jewelry by category, gender, and material
- Shop discovery and vendor profiles
- Shopping cart and order management
- Custom jewelry orders
- Video calling with vendors
- Profile and address management
- Live gold rate tracking

## Technology Stack

- HTML5, CSS3, JavaScript (ES6+)
- Firebase Authentication
- WebRTC for video calls
- Socket.IO for real-time communication

## Setup

1. Open `index.html` in a web browser
2. Configure Firebase credentials in `js/config.js`
3. Ensure backend API is running at the configured URL

## API Backend

Backend URL: `https://swarna-setu-api.onrender.com`
