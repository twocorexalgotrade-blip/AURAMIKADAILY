# AURAMIKA DAILY

**Premium Imitation Jewelry for the Modern Indian Woman**

> Luxury-look. Real-feel. Prices that make sense.

---

## 🌐 Live Site
[auramikadaily.onrender.com](https://auramikadaily.onrender.com)

## 🛍️ Features
- Full e-commerce website built with **Next.js 15 + Tailwind CSS**
- Shop page with product filtering & sorting
- Product Detail Pages (PDP) with image gallery, reviews, pincode delivery check
- Cart with coupon codes & order summary
- 2-step Checkout with **Cashfree Payments**
- Order success page with tracking
- Collections, Gifting, About, FAQs, Contact, Returns, Shipping Policy pages
- Fully mobile-responsive

## 🚀 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Payments**: Cashfree Payments SDK
- **Hosting**: Render

## 🔧 Running Locally
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🏗️ Build & Deploy
```bash
npm run build
npm run start
```

## 🔐 Render Deployment
- **Build Command**: `npm install && npm run build`
- **Start Command**: `node .next/standalone/server.js`
- **Environment**: Node.js
- **Health Check**: `/`
